Swapping Of Two Numbers


In this program we will swap two numbers.


➣ Source Code

 
		   
#include <stdio.h>
int main()
{
      int firstNumber, secondNumber, temp;

      printf("Enter first number: ");
      scanf("%d", &firstNumber);

      printf("Enter second number: ");
      scanf("%d",&secondNumber);

      // Value of firstNumber is assigned to temp
      temp = firstNumber;

      // Value of secondNumber is assigned to firstNumber
      firstNumber = secondNumber;

      // Value of temp(which contains the initial value of firstNumber) is assigned to secondNumber
      secondNumber = temp;

      printf("\nAfter swapping, firstNumber = %.2lf\n", firstNumber);
      printf("After swapping, secondNumber = %.2lf", secondNumber);

      return 0;
}

➣ Output

 
Enter first number: 46
Enter second number: 69

After swapping, firstNumber = 69
After swapping, secondNumber = 46
